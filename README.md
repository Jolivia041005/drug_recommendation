# 2型糖尿病（T2DM）治疗用药推荐

基于《中国糖尿病防治指南（2024版）》及《成人2型糖尿病口服降糖药联合治疗专家共识（2025版）》开发的临床决策支持工具，为医务工作者提供个体化用药推荐、药物对比、肝肾剂量调整及相互作用检查的一站式服务。

## 功能概览

- **药物信息浏览**：按药物分类查看所有降糖药的详细资料（作用机制、靶点、降糖效力、低血糖风险、体重影响、心肾获益、禁忌症等），并显示分子结构图。
- **药物对比分析**：选择2～4种药物，多维度对比关键属性，并生成能力雷达图（降糖效力、低血糖安全性、体重获益、心肾获益）。
- **个体化用药推荐**：基于患者临床特征（年龄、BMI、HbA1c、合并症、高危因素、治疗阶段等），依据指南推荐路径输出单药/二联/三联方案，并自动匹配固定剂量复方制剂（FDC）简化方案。
- **肝肾功能剂量调整**：输入eGFR和Child-Pugh分级，根据药品肝肾功能需求提示信息，判断每种药物是否可用、减量慎用或禁用，并以颜色标识。
- **药物相互作用检查**：选择患者当前用药和身体状况，识别已知相互作用（如二甲双胍与碘化对比剂、SGLT2i与糖皮质激素、沙格列汀与心衰等），并提供临床建议。

## 技术栈

- Python 3.8+
- Streamlit（Web框架）
- Pandas（数据处理）
- Plotly（雷达图可视化）
- 数据文件：CSV格式
